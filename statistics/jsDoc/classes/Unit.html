Unit

Defined in js\classes\Helper\Unit.js:201

Sums up the target and unit.

If they have different units of measurement, an error is thrown unless at least of the operands is different from zero.

Examples:

  1. 1cm + 2cm = 3cm

  2. 1cm + 2px = error

  3. 4 + 7 = 11

  4. 0px + 2cm = 2cm

  5. 1px + 0cm = 1px

  6. 0 + 2cm = 2cm

  7. 1px + 0 = 1px

  8. 0px + 0cm = 0

  9. 0 + 0cm = 0

  10. 0px + 0 = 0

  11. 0 + 0 = 0

constructor

(
  • value
  • measure
)

Defined in js\classes\Helper\Unit.js:37

Initializes properties _value and _measure. The order is as follows:

  1. If value is an instance of Unit, then its value and measure are used to initialize corresponding properties of the current instance.
  2. In case measure is provided and it is not an empty string, its value is assigned to _measure.
  3. If value is a number, then _value is set to that value.
  4. If value is a string, it is parsed into a float number which is then assigned to _value. Then it is tried to pick up eventual information about dimension from that string and if the dimensiomnturns out to be non-empty string and _measure turns out to be not initialized, the found dimension is assigned to _measure.

Parameters:

  • measure String | Null

frac

  • u
  • p
)
Unit

Defined in js\classes\Helper\Unit.js:331

Returns the result of division of the target by the argument.

The following cases are distinguished (in order of processing):

  1. If the argument is a non-zero number, then _value of the target is divided by this number. If it is zero, an error is thrown.
  2. If the argument is an instance of Unit:

    • if the denominator is dimensionless or has the same dimensionality as the numerator, then division is performed normally
    • if the numerator and denominator have different dimesions, then an error is thrown.
If the argument is not an instance of Unit class, then the method is called again with argument being converted into Unit instance.

Parameters:

  • u Unit
  • p Integer

    Precision (optional). Number of digits after the decimal point
