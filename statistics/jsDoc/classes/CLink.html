Collects parameters from link dialog menu.

The returning object include the following keys:

href
(string) url or email (depending on a scheme)
text
(string) hyperlink text
status
(boolean) whether text field is enabled
title
(string, optional) popup text
underlined
(boolean) whether the link is underlined or not
target
(boolean) whether the link should be open in a new window
color
(string) color of the hyperlink text
