Document Class

Defined in: js\classes\HtmlElements\Document.js:4
Module: Document

This class is to deal with documents: parsing, converting, saving. Its functionality is similar to those of Dom.

The difference between them: Document is a singleton, while Dom is a class that accepts creation of multiple instances.
