knockOutCell

(
  • cellNum
)
Void

Defined in js\classes\HtmlElements\Tag\Row.js:130

Drops the cell at the given position and resizes the remaining cells. If the cell to drop has a nieghbour to its right, then the freed space isassigned to that neighbour, otherwise it is assigned to the left neighbour:

|xxx| a | b   | c |   ->   |     a | b   | c |
| a |xxx| b   | c |   ->   | a |     b   | c |
| a | b | c | xxx |   ->   | a | b | c       |
If the cell to drop does not exist, the row remains unchanged.

Parameters:

  • cellNum Number

    cell number to delete. Numeration starts with 0.

Returns:
