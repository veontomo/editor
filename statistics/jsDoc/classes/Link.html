apply

(
  • obj
)
Any

Defined in js\classes\HtmlElements\Tag\Link.js:89

Apply target properties on the argument obj. The following cases are distinguished:

  • the argument is an instance of Link. In this case, returns output of updateLink method.
  • the argument is an instance of Content. In this case, returns output of applyContent method.
  • the argument is an instance of Tag. In this case, returns output of applyTag method.
If non of the above holds, result of wrap method is returned.

applyTag

(
  • tagObj
)
Tag | Link

Defined in js\classes\HtmlElements\Tag\Link.js:144

Converts the argument in link. Argument tagObj must be a Tag instance. If its Tag/content is

  1. non empty, then it is returned a copy of the argument in which content is replaced by output of applyContent method.
  2. empty, then a Link instance is returned. This instance has Tag/attributes:property and Tag/styles:property of the target and its content contains tagObj as the only element.

Defined in js\classes\HtmlElements\Tag\Link.js:60

Sets text-attribute of the Link/style:property property. If the argument is missing or equal to true, "underline" is imposed. If the argument is false, then "none" is imposed. If the argument is a string, text-property will be assigned to be equal to that string. If nothing of the above holds, text-property remains unchanged.
