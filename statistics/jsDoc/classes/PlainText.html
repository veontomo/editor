PlainText Class

Defined in: js\classes\HtmlElements\PlainText.js:5
Module: HtmlElements

Represents plain text. This class is intented to represent text node elements. Though there is no text tag in HTML, it is introduced here in order to consider plain text on the same basis as other tags.

clone

() Object

Defined in js\classes\HtmlElements\PlainText.js:149

Clones the target. Parses all attributes of the target. If the attribute responds to "clone" method, then assign the result of this method to the corresponding clone attribute. Otherwise, assign the attribute value to the clone attribute (potentially problematic for what is passed by reference and not by value, like arrays). content is a private variable, so in the clone it is fed by means of getContent() and setContent() methods.
