toHtml

() String

Defined in js\classes\HtmlElements\Tag\Image.js:179

Returns html representation of the instance if getOrigin() returns non-empty string. Otherwise, returns empty string.

Html representation consists of opening and closing tags that are output of methods openingTag and closingTag correspondingly.

This method overrides the parent one toHtml (since I could not consistently call a parent class method from a child one when the child class overrides the corresponding parent method.)
