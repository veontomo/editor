Converts each element of this instance into a node and appends it to the argument.

Takes each element of the array Content::elements, converts it into a node (if the element responds to toNode() method, the convertion will be performed using this method, otherwise a text node will be constructed) and then appends this node to the argument which is supposed to be an instance of DOM.Node (in fact, it is enough that is has appendChild() method).
