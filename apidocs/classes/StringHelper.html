createListFromHtml

(

)
List | Null

Defined in js\classes\Helper\String.js:155

Transforms a list string into a List object. The argument listType is used to initialize the property "name" of the returned instance. If it is not provided, that the node tag-name property is used. It is supposed that the string to process is of the following form: <ol ... > ... </ol> or <ul ... > ... </ul>. Inside the tag, there might be other nodes. If they are recognized as a "supported" ones, the corresponding functions will be called to transform them into objects.
