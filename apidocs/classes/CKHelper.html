bunch-next-siblings

(
  • node
  • root
)
Array

Defined in js\classes\CKHelper.js:463

Returns an array containing node and elements that come after it the in DOM in the context of root. Therefore, all array elements belong to root. root itself is not included in the output except the case when it is equal to node. If root does not contain node, the output must be an empty array. Uses CKHelper['next-siblings'] to fill in array with the next siblings.

Parameters:

  • node CKEDITOR.dom.element | CKEDITOR.dom.node

    a node that must be inside of root node

  • root CKEDITOR.dom.element | CKEDITOR.dom.node

    the returned array elements will be inside this node.

Returns:

Array:

nodes between node and root last child (inclusively)

doesOverlap

(
  • elem1
  • elem2
)
Boolean

Defined in js\classes\CKHelper.js:325

Check whether two different CKEDITOR.dom.node's "overlaps". Returns true, if:

  1. both elem1 and elem2 are of CKEDITOR.dom.element type and either elem1 contains elem2 or viceverse,

  2. only one of elem1, elem2 is of CKEDITOR.dom.element and it contains other,

  3. both elem1 and elem2 are of CKEDITOR.dom.text and they are equal.
Otherwise, false is returned.

Parameters:

  • elem1 CKEDITOR.dom.element | CKEDITOR.dom.text
  • elem2 CKEDITOR.dom.element | CKEDITOR.dom.text

Gets the string representation of the node subelement: if the argument is of CKEDITOR.NODE_ELEMENT type, then the string representation of its child "offset" is returned.
