insertRow

(
  • ed
  • pos
)

Defined in js\classes\CKHelper.js:56

Inserts a row at a specified position with respect to the selected element. The command to insert the row is obtained by capitalizing the second argument and appending it to the string 'insert'. Example: if pos is 'after', the command to be executed is 'insertAfter'.

Parameters:

  • ed CKEDITOR.editor

    Represents an editor instance.

  • pos String

    where to insert the element with respect to the current one.
