String

Defined in js\classes\Properties\Styles.js:40

Returns "bare" string of key-values pairs of style properties, i.e. "width: 10px; color: #010101". It takes into consideration core keys only. If value corresponding to the key is a number, then the measurement unit will be appended. If core is empty, then empty string is returned.

Parameters:

  • unit String | Null

    measurement unit to be added to the numerical attribute values. By default, it is set to 'px'.
