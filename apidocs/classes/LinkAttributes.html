Loads attributes from the argument that is supposed to be of a type NamedNodeMap. Nevertheless, it is sufficient that attr be a collection of objects with name and value properties. If among the attributes there is node with name "style", it gets ignored. If it is attempted to override a method and not a property, then false is returned.
