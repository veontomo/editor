  2. it contains a single element that has type text

selectedNodes

(
  • sel
)
Array private

Defined in js\classes\CKHelper\Selection.js:106

If selection is empty, returns empty array. Otherwise returns two dimensional array of the form

[[a00, a01, ...], [a10, a11, ...], ...].
Each inner array corresponds to the elements inside the ranges property of the selection. Since DOM is an ordered collection of the nodes, the the above mentioned array is just a collection of simply-connected sets of nodes corresponding to the selection.

NB1: Simply-connected set is a set such that there exists a path inside the set connecting two arbitrary elements of the set.

NB2: Path consists of pieces connecting two neighbours (the set is ordered, so that the concept of "neighbour" exists).

Parameters:

  • sel Selection

Returns:

Array:

two dimensional array of CKEDITOR.dom.domObject or empty array
