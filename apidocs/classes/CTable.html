parentWidth

() Number

Defined in js\controllers\CTable.js:13

Returns the width of the parent element available for its children.

available width = (element width) - (element left border width) -
        (element right border width) - (element left margin) - (element right margin)
The element width is supposed to be greater than zero and hence to have a unit of measurement (e.g. 'px'). If not set, widths of other attributes are equal to zero without unit of measurement. In this case one has to set the unit of measurement equal to the element width.

Returns:

Number:

available width for the children as Unit object (with properties "value" and "measure")
