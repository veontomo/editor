Unit

Defined in js\classes\Helper\Unit.js:186

Returns the result of division of the target by the argument.

The following cases are distinguished (in order of processing):

  1. If the argument is a non-zero number, then Unit/value:property of the target is divided by this number. If it is zero, an error is thrown.
  2. If the argument is an instance of Unit:

    • if the denominator is dimensionless or has the same dimensionality as the numerator, then division is performed normally
    • if the numerator and denominator have different dimesions, then an error is thrown.
If the argument is not an instance of Unit class, then the method is called again with argument being converted into Unit instance.
