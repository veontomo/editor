clone

() Object

Defined in js\classes\HtmlElements\Tag.js:753

Creates a clone of the target. If the target has a className attribute corresponding to exisiting class, then that class instance is to be returned. Otherwise a Tag instance is returned. The method parses all attributes of the target and if the attribute responds to a "clone" method, then calls this method and assigns its result to the corresponding clone attribute. Otherwise, assign target attribute value to the clone attribute (there might be a potential problem with what is passed by reference and not by values, i.e. array). But among Tag properties there are no array-valued ones.

Returns:

Object:

load

(
  • elem
)
Boolean

Defined in js\classes\HtmlElements\Tag.js:781

Populates the following properties of the target object

  1. tag
  2. attr by calling Attribute::load() method.
  3. style by calling Styles::load() method.
  4. content by calling Content::load() method.
from the argument which must be an instance of Element class. Returns true if the argument is of required type and the above mentioned load methods return true. Otherwise, false is returned.
NB: DOM.Element.attributes has the form {1: {tag: "width", value:"100", ...}, 2: {tag: "color", value:"black", ...}, ...}

Parameters:

  • elem Element

    origin from which the element properties are to be loaded

Returns:

Boolean:
