Properties Class

Defined in: js\classes\Properties.js:5
Module: Properties

A general Property class. If the argument is an object, then its properties are copied into Property instance. If the argument is a string, then it will be splitted according to the pattern "key: value;" to populate object properties.

Clones the target. Parses all attributes of the target. If the attribute responds to "clone" method, then assign the result of this method to the corresponding clone attribute. Otherwise, assign the attribute value to the clone attribute (potentially problematic for what is passed by reference and not by value, like arrays). Core is a private variable, so in the clone it is populated by means of getProperty() and setProperty() method.

toggleProperty

(
  • propName
  • val
  • altVal
)
Void

Defined in js\classes\Properties.js:267

Switches property propName between val and altVal in the following way:

  1. if the instance has property propName equal to val, then the method imposes its value to be altVal (if altVal is not defined, then the key propName gets dropped).
  2. if the instance contains property propName equal to altVal, then the method sets its value to be val.
  3. if the instance does not have property propName, then the method imposes it to be val.

Parameters:

  • propName String

    property key to toggle (i.e., "width", "position")

  • val String

    value of property key (i.e., "25em", "fixed")

  • altVal String | Null

    alternative value of property key (i.e. null, "0em"). This parameter is optional.

mode

Integer private

Defined in js\classes\Properties.js:60

Mode to be used when representing this instance as a string by means of Properties/toString:method method. If it is set to 0, then string representation has the following form width="20" title="read me!" If it is set to 1, then string representation has the following form width: 50px; color: red.

Default: 0
