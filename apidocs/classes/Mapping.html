Mapping Class

Defined in: js\classes\HtmlElements\Mapping.js:5
Module: HtmlElements

This class associates different types of object. It is intended to be used along with Factory class. The idea is to have criteria on which decide what type of objects should be constructed. This class is responsible for these criteria.
