It takes a string that is an html representation of a table and update current object parameters such that it will correspond to the html representation. Inherited from Table: attributes.js:1122

Removes the border of the table. It updates the properties 'attr' and 'style' of the instance:

  1. in 'style' property, deletes the properties: 'border-width', 'border-color' and sets up 'border-style' to 'none'
  2. in 'attr' property, deletes 'border' property.

setBorder

(
  • borderInfo
)
Void

Inherited from Table: attributes.js:1098

Set the border of the table. It updates the properties 'attr' and 'style' of the instance:

  1. in 'style' property, sets up the following properties: 'border-width', 'border-color' and 'border-style'
  2. in 'attr' property, sets up 'border' property. Note that if after setting the border there is an assigment of 'style' or 'attr' property, then some info about the border might be overwritten.

Parameters:

  • borderInfo Object

    Object containing 'width', 'color' and 'style' fo the border to set.
