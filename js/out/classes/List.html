  • item
)
Void

Defined in list.js:142

Inserts the item at the given position. If the list contains N items, then allowed index for the item to be inserted at is the range [0, N]. Zero index corresponds to insertion at the beginning of the list, N - to the end of the list, that is to appending (delegates to List::appendItem). If the index is out of this range, an error is thrown. If the item to be inserted is not an instance of ListItem, an error is thrown.

Parameters:

  • pos Number

    index of the position of there to insert the item
